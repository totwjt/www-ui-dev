## 版本依赖

`vue@3.3.13`

`ant-design-vue@3.2.15`

`pnpm@7.x`

`node@v16.19.0`

### 安装

使用 npm 或 * 安装 `www-ui` 组件库。

```bash
npm run login:aliyun

# install
npm install www-ui
```

## 全局引入

```js
import wwwUi from 'www-ui';
app.use(wwwUi);

```
